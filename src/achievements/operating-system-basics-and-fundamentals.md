Successfully achieved the student level credential for completing the 'Operating Systems Basics' course through the Cisco Networking Academy. This certification demonstrates my proficiency in both Windows and Linux operating systems, including:

- Explaining the architecture of Windows and its operation
- Using Windows administrative tools to configure, monitor, and manage system resources
- Using the Linux shell to manipulate text files, identify services, and locate and monitor log files
- Implementing basic Linux security protocols
- Configuring network connectivity and email on mobile devices
- Understanding the purpose and characteristics of mobile operating systems

This course equipped me with practical, hands-on skills essential for modern IT infrastructure administration and system management.
