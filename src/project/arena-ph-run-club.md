<article>

ARENA.PH Run Club is a premier Filipino running community platform dedicated to connecting runners across the Philippines. From the scenic routes of UP Diliman and BGC to the challenging trails of Timberland, this platform celebrates the local running culture and lifestyle — bridging the gap between solo running and community excellence.

<br/>

**Vision**

Built for Filipino runners by a Filipino developer, ARENA.PH Run Club serves as the digital home for a growing grassroots movement. The platform is designed to help runners find their tribe, access community-exclusive events, and achieve personal bests with the support of a passionate, nationwide network.

<br/>

**Key Value Proposition**

Join the club to find your tribe, access community-exclusive events, and achieve your personal best with the support of the ARENA.PH community.

<br/>

**Community Stats**

- 500+ Filipino Runners across the nation
- 8+ Cities covered from Metro Manila to the provinces
- 52+ Weekly Group Runs keeping the community moving
- 200+ PRs Broken and Goals Achieved by club members

<br/>

**Key Features**

- Community Hub: A central space for Filipino runners to connect, share experiences, and motivate one another.
- Event Discovery: Browse and join community-exclusive group runs, races, and training sessions across the Philippines.
- Local Route Highlights: Curated running routes from iconic spots like UP Diliman, BGC, and Timberland trails.
- Member Achievements: Track and celebrate personal records and milestones within the community.
- Responsive Design: Optimized for mobile and desktop, ensuring runners can connect on the go.

<br/>

**Technical Challenges & Solutions**

- Real-Time Community Data: Leveraged Firebase Realtime Database to keep community stats, event listings, and member milestones synchronized instantly across all users.
- AI-Powered Engagement: Integrated a chatbot to handle community FAQs and guide new members, reducing onboarding friction.
- Performance on Mobile: Implemented Next.js image optimization and lazy loading to maintain fast load times for runners accessing the platform on mobile networks.

<br/>

**Impact**

- Connected 500+ Filipino runners under a single, organized platform.
- Facilitated 52+ weekly group runs across 8+ cities in the Philippines.
- Fostered a culture of community accountability that has led to 200+ personal records broken and goals achieved.

<br/>

**Architecture**

- Frontend: Next.js with TypeScript for a performant, type-safe, and SEO-optimized web application.
- Styling: Tailwind CSS for a clean, responsive, and mobile-first design system.
- Backend Services: Firebase for real-time database, authentication, and cloud storage.
- AI Integration: Conversational chatbot layer for community support and FAQs.
- Deployment: Hosted on Vercel for zero-downtime continuous deployment with global edge distribution.

This project reflects my commitment to building technology that serves real, local communities. Working on ARENA.PH Run Club deepened my expertise in full-stack Next.js development, Firebase integration, and designing community-first user experiences for a Filipino audience.
